Preface

There is time when you need just simple mock backend for your web application or experiment. That's why I've developed DummyAPI.

Features

  1. REST-like interface
  2. Socket.IO notifications (TBD)
  3. Proxy functions for data transformations before sending to client
  4. Mock authentication with users and user groups
  5. Custom resource routes
  6. ...
